# colorify Logs


## Installation

```bash
yarn add toast-logs
```


## Methods

```js
import { Log } from 'toast-logs'
```

## Example

```js
 Log.success('Success message')

 Log.danger('Error message ')
 
 Log.info('Information message')
```
